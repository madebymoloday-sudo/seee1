# Программы психолога

Эта папка содержит JSON конфигурации для разных программ работы психолога.

## Структура

Каждый файл JSON представляет отдельную программу психолога с уникальным пайплайном вопросов.

## Формат файла

```json
{
  "name": "название программы",
  "version": "1.0.0",
  "description": "Описание программы",
  "steps": {
    "step_name": {
      "question": "Вопрос для пользователя (или null)",
      "systemPrompt": "Промпт для GPT с переменными {variable}"
    }
  },
  "variables": {
    "variable": "Описание переменной"
  },
  "settings": {
    "temperature": 0.7,
    "maxTokens": 500,
    "model": "gpt-4o-mini"
  }
}
```

## Переменные в промптах

В `systemPrompt` можно использовать переменные в фигурных скобках:
- `{problem}` - проблема пользователя
- `{emotion}` - эмоция пользователя
- `{thought}` - мысль пользователя
- и т.д.

## Добавление новой программы

1. Создайте новый JSON файл в этой папке
2. Укажите уникальное имя программы
3. Определите шаги пайплайна
4. Программа будет автоматически доступна после перезапуска сервера

## Примеры программ

- `default.json` - стандартный 9-шаговый пайплайн
- Можно добавить `short.json` - сокращенный пайплайн
- Можно добавить `deep.json` - углубленный пайплайн
